# 超市财务管理系统

一个基于Python Flask框架开发的超市财务管理系统，提供商品管理、供应商管理、采购管理、销售管理、财务管理等功能。

## 功能特点

- **用户认证**：支持用户注册、登录和角色管理
- **商品管理**：添加、查看、编辑和删除商品信息
- **供应商管理**：管理供应商信息
- **采购管理**：记录商品采购信息，自动更新库存
- **销售管理**：记录商品销售信息，自动减少库存
- **库存管理**：实时跟踪商品库存状态
- **财务管理**：记录费用和收入，生成财务报表
- **数据统计**：提供销售、利润等统计数据

## 技术栈

- **后端框架**：Flask
- **数据库**：SQLite（使用SQLAlchemy ORM）
- **前端技术**：HTML5、Bootstrap 5、Jinja2模板
- **认证系统**：Flask-Login
- **密码加密**：Flask-Bcrypt

## 安装和运行

### 1. 安装依赖

```bash
pip install flask flask-sqlalchemy flask-bcrypt flask-login
```

### 2. 启动应用程序

```bash
python app.py
```

### 3. 访问系统

打开浏览器，访问 http://localhost:5000

## 系统使用说明

### 1. 用户管理

- 管理员可以创建新用户并分配角色
- 普通用户可以修改自己的密码

### 2. 商品管理

- 添加新商品时需要输入商品名称、类别、售价和成本价
- 系统会自动为新商品初始化库存为0
- 可以随时编辑或删除商品信息

### 3. 供应商管理

- 管理供应商信息，包括名称、联系人、电话和地址
- 供应商信息用于采购记录

### 4. 采购管理

- 记录商品采购信息，包括商品、供应商、数量
- 系统会自动计算总成本并更新库存

### 5. 销售管理

- 记录商品销售信息，包括商品和数量
- 系统会自动计算总金额并减少库存
- 如果库存不足，系统会提示无法完成销售

### 6. 财务管理

- 记录费用和收入信息
- 提供费用和收入的分类统计

### 7. 报表分析

- 销售报表：查看销售数据统计
- 库存报表：查看商品库存状态
- 财务报表：查看财务数据统计

## 注意事项

1. 系统使用SQLite数据库，数据文件会自动创建在应用程序目录下
2. 默认管理员账户密码为admin123，请及时修改
3. 系统会自动计算库存和财务数据，请确保输入数据的准确性
4. 如果需要修改数据库配置，可以在app.py文件中修改相关配置项
